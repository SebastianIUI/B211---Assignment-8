Purpose: The purpose of this assignment is to start performing data visualization using python and the Seaborn library, using real world data.

Class Design/Implementation: 
X: Used for numerical measurements describing characterisitcs for each datset. For the exercise dataset, the numerical variables include the pulse measurements taken at 1 minute, 15 minutes, and 30 minutes after exercise. For the planets dataset, numerical values such as planet mass, orbital period, and distance are used to analyze relationships and distributions in the data

Y: Used for categorical values representing the class of each observation. For the exercise dataset, the categorical variables include diet type and type of exercise, which allow comparisons of pulse rates across different groups. In the planets dataset, categorical variables such as the method used to discover the planet help group planets and compare how different discovery techniques contribute to planet detection.

Data Cleaning: The data from the CSV file was processed to ensure it could be properly used for visualization. This included organizing the pulse measurements into a structured format that allowed comparisons across time intervals, diets, and exercise types.

Plotting Functions: he visualizations were created using Seaborn along with Matplotlib to generate graphs with clear axes, titles, and legends. Seaborn was used to create the heatmap, relational plots, distribution plots, and categorical plots.

Limitations/Improvements:
- The analysis is limited to the variables available in the datasets which restricts deeper insights.
- An improvement would be adding more advanced visualizations via combining multiple plots into a subplot for easier comparison.
